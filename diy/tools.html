Demolition

  • Sledge hammer
    • With more weight, you won't have to swing as fast - good ref. In the pallets video, 3 pounds (lb) was recommended. $1lb = \frac{1}{2.205}kg$. Fibreglass handle is better at being shock absorbant.

  • Prybar
    • I went for a 15inch (38.1cm) length. Apparently, it's got an extra wide blade which should prevent marring. Useful link on prybars.

Drill bits

Erbauer drill bits:
  • 13 x brad point wood drill bits (3, 3.2, 3.5, 4, 4.5, 4.8, 5, 5.5, 6, 6.5, 7, 7.5 and 8mm)

Sandpaper

  • Grit size
    • Ref - Lower the number, larger the grit size, coarser the sandpaper. I would need a fine (120-220 grit) sandpaper to sand off the edges of the venetian blinds.

  • Grit material
    • I went for one with synthetic grit made of Aluminium oxide. It can be used to sand metal as well.

Hacksaw

  • Tooth per inch
    • This link recommends 32-teeth per inch for cutting light metals like aluminium. Blades with more teeth will provide a smoother, finer cut.

  • Demonstration of use
    • This link recommends not leaving a blade tightened up when storing away. Also, a hacksaw only cuts in the forward motion i.e. the direction in which the blade tooth are pointing.

  • Speed of cut vs TPI
    • Ref - less teeth, more time to cut, rougher finish.

Ladders

I like this video on combination ladders. The floor to ceiling height to the attic door = 2.6m. This ladder in the extension mode has a height of 3.51m. Closed height of 1.71m.
