Overview

At each step A* picks the node having the lowest $f$ which is a parameter equal to the sum of two other parameters:
  • $g$: the cost to move from the starting point to a node under consideration, following the path generated to get there
  • $h$: the estimated cost to move from that node to the final destination. This is referred to as the heuristic, which is nothing but a kind of smart guess. We don't know the actual distance until we find the path because all sorts of obstructions can be in the way. There can be many ways to calculate $h$.
