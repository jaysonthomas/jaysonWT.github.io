Bazel: Concepts

Workspace, Packages

A workspace is the root folder that contains all the data that is needed to build a project. It is identified by a WORKSPACE or WORKSPACE.bazel file. It is represented by // in a path.

The workspace is subdivided into packages. Each folder of the workspace can act as a package (including the directory that contains the WORKSPACE file) and identified by a BUILD or BUILD.bazel file.
