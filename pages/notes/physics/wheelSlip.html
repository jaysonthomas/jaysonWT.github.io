Types of friction

Static friction

If an object is at rest w.r.t. a surface, we can apply horizontal forces up to a certain threshold force and the object will still remain static. Experiments show this threshold force is a constant times the normal force. The coefficient is called the coefficient of static friction. $$F_s>\mu_s*N$$ $$\frac{1}{3} < \mu_s < 1$$ This means that the force required to move the object horizontally is always less than the force with which gravity pulls the object down.
