Arm: Inverse Kinematics

Description

Inverse kinematics is a nonlinear complicated function that helps to go from poses in the world to joint positions. $$ q = (f^B)^{-1}(X^B) $$ It is not necessarily a unique mapping - there can be multiple $q$s that get the same pose in the world. There could also be no solutions. Going the other way is more straight forward.

IK is oftentimes not used directly since if the IK engine gives slightly different answers for very similar poses, then it might involve arm movements that jump around to different solutions. It is normally used with additional requirements for continuity/smoothness/centering etc.

An example: if an arm had multiple joint angle solutions for a given pose for the hand, then the IK engine could ask for different joint angles (for the same hand pose) every 0.0001 seconds which wouldn't be good.
