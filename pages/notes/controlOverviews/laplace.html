Frequency domain and the fourier transform

  • Representing mass spring system in frequency vs time domain - mathematical model
  • Fourier transform - mapping from time to freq domain. Amplitude, phase, freq.
  • Multiply 2 freq domain signals [input * impulse response] to get the system response == Convolution in time domain.
  • Multiply step with exponential decay signal. Taking fourier transform gives a complex function that is made up of 2 numbers - a real and an imaginary component.
  • Separate out the 2 parts and change the representation from rectangular to polar coordinates to find the phase and magnitude of the resulting signal that is a function of frequency.
  • 1D signal in time domain is converted to 2D signal in frequency domain with real and imaginary components; which, through some additional algebra, are the magnitude and phase of the cosine waves that make up the original time domain function.

s domain and the laplace transform

  • Instead of just cosine waves (fourier transform), the Laplace transform decomposes a time domain signal into both cosines and exponential functions. A symbol that represents both frequency and the exponential component is needed $s = \sigma + j\omega$.
  • An exponential signal with an imaginary exponent is a sinusoidal signal. An exponential signal with a real exponent either grows (positive exponent) or decays (negative exponent) exponentially.
  • The s plane is composed of a horizontal and a vertical axis representing the real and the imaginary component respectively.
  • Laplace transform transforms time to s domain which is 3 dimensional - $\sigma, j\omega, \omega$.
  • So much interest in representing all signals as function of exponential and sinusoids because these 2 don't change as the 6 allowed LTI operations are applied.

  • We use a FBD to identify and sum forces acting on a system. Then, derive the equations of motion (2nd ODE), that we can solve for the state given an input force. This process of creating a model of the system is called system identification.
