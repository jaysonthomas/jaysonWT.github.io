Gaussians

Background

[Ref] The basic form is given as: $$ f(x) = e^{-x^2} $$ The familiar gaussian equation used in probability is given as: $$ p(x) = \frac{1}{\sqrt{2\pi \sigma^2}} e^{\frac{-(x-\mu)^2}{2\sigma^2}} $$

Equation breakdown

[Desmos plots]
  • $e^{x}$
    • $e^{0.5} = 1.648$
    $e^{1} = 2.718$
    $e^{5} = 148.413$

    With positive exponents, the value of the function keeps increasing exponentially.

    $e^{-0.5} = 0.606$
    $e^{-1} = 0.367$
    $e^{-5} = 0.006737$

    With negative exponents, the value keeps decreasing exponentially.

  • $e^{-x}$
    • This is a mirror of $e^x$. The value of the function decreases exponentially with positive $x$. But, increases exponentially with negative $x$. To get the familiar bell shape, we need to curb this exponential increase with negative values of $x$ and replicate the exponential decrease seen with positive $x$. Hence, we square $x$.

  • $e^{-(x-\mu)^2}$
    • The peak of the bell curve gets centred on $\mu$, i.e. with increasing positive values of $\mu$, the peak shifts to the right.

  • $e^{-\frac{(x-\mu)^2}{\sigma^2}}$
    • As $\sigma^2$ increases, the exponent becomes less negative, i.e. the exponent increases and the overall curve becomes flatter at the peak. For a given value of $x$, the function's value is now larger, so the curve doesn't have to decrease as fast.

    To increase the probability of an event, we would need to decrease the variance, i.e. for a given value of $x$, the function's value is now smaller, so the curve has to decrease faster. The curve would be less flat at the peak.

    If $\sigma$ wasn't squared, we would get an inverted bell curve.
