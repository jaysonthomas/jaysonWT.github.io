Jacobian

As seen in different contexts

  • Transformation of a basis vector
  • Two-link Arm

Description

Each point in an ($n$ dimensional) output space is created by $m$ contributing vectors (of $n$ dimensions each). Each of these vectors is uniquely influenced by one input space variable. There are $m$ input space variables.

A Jacobian captures how a change in any input space variable affects the output space. Each column of the Jacobian captures how a change in an input space variable affects its corresponding (influenced) $n-dimensional$ vector in the output space. So, a Jacobian has $n$ rows and $m$ columns.

For example, consider a planar 1R Arm. The input space would consist of angle $\theta$ and the output space would consist of 1 2-dimensional point in $x, y$. The Jacobian would be given by: $$\begin{bmatrix} \frac{\partial{f_x(\theta)}}{\partial{\theta}} \\ \frac{\partial{f_y(\theta)}}{\partial{\theta}} \end{bmatrix}$$ $f_x(\theta), f_y(\theta)$ alone could give the change in the output space due to a change in an input space variable, but it is divided/normalised by the change in the input space variable ($\partial{\theta}$) to find simply the constant change that is not affected/scaled by the amount of change in the input space.

If the input space consisted of 2 variables $\theta_1, \theta_2$, then the Jacobian would be: $$\begin{bmatrix} \frac{\partial{f_x}}{\partial{\theta_1}} & \frac{\partial{f_x}}{\partial{\theta_2}} \\ \frac{\partial{f_y}}{\partial{\theta_1}} & \frac{\partial{f_y}}{\partial{\theta_2}} \end{bmatrix}$$

Statement 1

The Jacobian matrix, $H$, contains the partial derivatives of the measurement equation with respect to the parameter(s) of interest.

The Jacobian is essentially a matrix of partial derivatives. Each element of the Jacobian matrix represents the rate of change of the measurement equation with respect to a specific parameter. Statement 2

Each element of the Jacobian matrix represents how much the error changes when you make a small change in one of the parameters.

When you're performing a least squares fit, you aim to minimize the error between the model's predictions (your measurement equation) and the actual data points. To minimize this error, you need to adjust the parameters (θ) in your model. The Jacobian helps with this adjustment by telling you how sensitive the error is to changes in each parameter. Specifically, it quantifies how much the error changes when you make small adjustments to each parameter in your model.

In other words, by looking at the partial derivatives in the Jacobian matrix, you can figure out which parameters have the most influence on the error, and you can adjust those parameters in the right direction to minimize the error. So, the Jacobian serves as a key tool in the optimization process when you're trying to find the best-fitting parameters by iteratively changing them in a way that reduces the error between your model and the data.
