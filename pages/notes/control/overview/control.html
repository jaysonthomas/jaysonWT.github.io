Control systems overview

Definition and purpose

A control system refers to the method used:
  • To control and regulate physical processes through the use of feedback loops
  • With the goal of maintaining a desired performance
  • While compensating for disturbances, changes in system dynamics and environment, or uncertainties in the system/environment

In a perfect world, a model of a system's dynamics would be the only thing required. It would help design the system's inputs [motor turn rates for example] that result in a desired behaviour and the controls problem would be solved.

[Point 3 above] In the real world, a control problem is all about managing the messiness of the real world, and making sure a system achieves a task in spite of sources that cause errors in achieving a target.

Parts of a controller

Block diagrams are used to visualise important information about a controller and the system it controls. Arrows represent signals and boxes represent functions or equations.
  • Plant/System
    • Physically, it represents the system being controlled. Synthetically, it encapsulates the model equations.

  • Actuators - provide input to the plant
  • Sensors - measure the state of the plant
  • Controller - processes sensor data and generates control signal to the actuators

Signals/Signal path:
  • Feedback path - sends sensor data back to the controller
  • Input example - commanded thrust

  • Output example - an actual target location
    • Output describes what can be measured about a system. If velocity could be measured, it would also be included in the output.

  • Reference example - a target set location

Types of controllers

Open loop

There is no feedback from the system. They are susceptible to disturbances and errors. Errors will be unnoticed and are able to grow unchecked.
There are two main types of control systems:
  • Open-loop systems
    • They rely solely on the input signal to control the output.

  • Closed-loop systems
    • They rely on feedback to compare the output with the desired performance and adjust the input accordingly. They can adapt to changes in the environment.
