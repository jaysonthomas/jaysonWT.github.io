Control systems overview

Control system

Definition and purpose

A control system refers to the method used to control and regulate physical processes through the use of feedback loops with the goal of maintaining a desired performance while compensating for disturbances, changes in system dynamics and environment, or uncertainties in the system/environment.

In a perfect world, a model of a system's dynamics would be the only thing required. It would help design the system's inputs that result in a desired behaviour and the controls problem would be solved. In the real world, the control problem is all about managing the messiness of the real world, and making sure a system achieves a task in spite of sources that cause errors in achieving a target.

Types

There are two main types of control systems:
  • Open-loop systems
    • They rely solely on the input signal to control the output. They are susceptible to disturbances and errors.

  • Closed-loop systems
    • They rely on feedback to compare the output with the desired performance and adjust the input accordingly. They can adapt to changes in the environment.

Components

  • Plant - the system being controlled
  • Actuators - provide input to the plant
  • Sensors - measure the state of the plant
  • Feedback path - sends the sensor data back to the controller
  • Controller - processes the sensor data and generates the control signal to the actuators
