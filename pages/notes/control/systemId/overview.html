System ID: Overview

Methods

White box method involves writing out equations of motion in the form of differential equations of motion. Black box method involves conducting system tests in order to find unknown variables.

A need for alternate mathematical model representations

A differential equation is great for solving for the response of a system but it doesn't lend itself very well to
  • Analysing how a system naturally behaves
  • Manipulating a system so it behaves a desired way
The two most popular alternate representations are:
  • Transfer functions
    • They are a Laplace domain representation of the system and are associated with the era of control techniques labeled classical control theory.

  • State space representation
    • It is a time domain representation, packaged in matrix form, associated with the era labeled modern control theory.

A choice
