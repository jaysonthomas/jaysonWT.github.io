The problem

The derivative term contribution to the total torque applied to the motor: $$\displaylines{ D_{cur} = \frac{(K_d*\Delta_{e}) + (t_{decay}*D_{prev})} {t_{decay} + T} = (K*\Delta_{e}) + \frac{D_{prev}}{1+\frac{T}{t_{decay}}}\\ = (K*\Delta_{e}) + \frac{D_{prev}}{T_f} }$$ $K$ : $\frac{K_d}{t_{decay} + T}$
$T_f$ (filter time?) : $1+\frac{T}{t_{decay}}$
$T$ : loop period
$\Delta_{e}$ : $e_{cur} - e_{prev}$
$e$ : $pos_{target} - pos_{measured}$

Expanding this to get the values in time instant 0, 1, 2: $$\displaylines{ D_0 = K*\Delta_e\\ D_1 = (K*\Delta_{e1}) + \frac{K*\Delta_e} {1+\frac{T}{t_{decay}}} = K\bigg[\Delta_{e1} + \frac{\Delta_e} {1+\frac{T}{t_{decay}}}\bigg]\\ D_2 = K\bigg[\Delta_{e2} + \frac{\Delta_{e1}} {1+\frac{T}{t_{decay}}} + \frac{\Delta_e} {(1+\frac{T}{t_{decay}})^2}\bigg] }$$ Question: How does $K$ and $\frac{1}{1+\frac{T}{t_{decay}}}$ contribute to the filtering of a signal? What is the intuitive way of seeing it?
