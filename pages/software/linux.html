Software installation, update, removal

  • When the server from which ubuntu pulls updates didn't work, I found this useful. Just find the best server and go for whatever is suggested.

  • Ensure package manager is up-to-date
    • 
sudo apt-get update && sudo apt-get upgrade

  • Removing a package
    • Explanation of the below is given here. 
    
sudo apt-get purge 
sudo apt-get autoremove
    The autoremove option removes packages that were automatically installed because some other package required them but, with those other packages removed, they are no longer needed. Sometimes, an upgrade will suggest that you run this command.
