Datatypes

A language needs to allow creation of one's own datatypes else it would have to cater to so many needs. Examples:
  • User defined datatypes
    • Class. We can have a class sedan and instantiate multiple objects for it - a blue and a red sedan for example. And it's all about relationships amongst these classes.
    • Vector

  • Primitives
    • These are the fundamental types. The standard library includes additional types such as std::size_t [Ref] and std::string.
