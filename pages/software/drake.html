Installation

First installation attempt

On 04.07.22, I did the following: 

sudo apt-get install python3.10-venv
python3 -m venv env
env/bin/pip install --upgrade pip
env/bin/pip install drake
Got this error: 

ERROR: Could not find a version that satisfies the requirement drake (from versions: none)
ERROR: No matching distribution found for drake
Pip version is pip-22.1.2. I followed the instructions shown in installation via pip.

Second installation attempt

  • Pip installation of manipulation
    • I could have just installed drake via pip, but installing manipulation covers it and lots of other extra packages. I had to install wheel, upgrade pip and then install manipulation again. 
    
python3.8 -m venv env
source env/bin/activate
which python
python3 --version

pip3 install manipulation
python3 -m pip install --upgrade pip
pip3 install wheel
    There's a comment as follows, here 
    
Drake publishes pre-compiled binaries as PyPI wheel packages.
  • Test if drake's been installed
    • 
python3 -c 'import pydrake.all; print(pydrake.__file__)'
Output: /home/jayson/P/mitManipulation/env/lib/python3.8/site-packages/pydrake/__init__.py
  • Had to install jupyter notebook
    • This is to be able to run the exercises. 
    
pip3 install notebook
  • Get graphs working
    • I had to do the following outside of the virtual environment to get graphs working. 
    
sudo apt-get install graphviz
  • Separate download of the manipulation repository
    • I'm going directly to the source so I can stay up to date with the latest changes to the pset files. 
    
cd ~/p/2_drakePythonTests/env/lib/python3.8/site-packages/pydrake/share/drake/tutorials/manipulation
git clone https://github.com/RussTedrake/manipulation.git .

Temporary way of running Drake locally

  • Activate virtual environment
    • 
source env/bin/activate
  • Deactivate virtual environment
    • 
deactivate
  • Run drake tutorial locally
    • 
python3 -m pydrake.tutorials
Output: Serving notebooks from local directory: /home/jayson/p/2_drakePythonTests/env/lib/python3.8/site-packages/pydrake/share/drake/tutorials
    index.ipynb gets served at http://localhost:8888/notebooks/index.ipynb#.

  • Finding index.ipynb
    • 
grep -r "index.ipynb" /folder/path
